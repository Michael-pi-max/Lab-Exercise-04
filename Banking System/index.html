The Program Does the Following:

  • Enable To Create Banks by specifying the name, lowest amount & interest rate
  • Enable To Create Users - enabling/disabling the interest service provided
  • Upon User Registration account numbers is automatically generated for the user
  • A User Can Log in Via UserName or Account Number, the former required to show id upon sign in
  • Users can check balance, deposit, withdraw and transfer money
  • Users can also view history of their transaction (including interest)
  • 1/10 of the br to be transferred is deducted when User transferring money to another user from different bank
